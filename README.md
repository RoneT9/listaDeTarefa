# Lista de Tarefa 📝

Este é um projeto simples de **lista de tarefas** desenvolvido com **HTML, CSS e JavaScript**.

## 💡 Funcionalidades

- Adicionar nova tarefa
- Marcar tarefa como concluída
- Remover tarefa
- Interface limpa e responsiva

## 🚀 Tecnologias usadas

- HTML5
- CSS3
- JavaScript (puro)
